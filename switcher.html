Switcher Plugin

This is a jQuery Plugin that toggles, or switches out, the content being displayed.

You can set any of the following options to customize it to fit your needs:


// options
$('selector').switcher({
  activeTargetClass: 'switcher-target-active', // class to add to the selected trigger
  activeContentClass: 'switcher-content-active', // class to add to the selected trigger
  callback: function(){},
  collection: null, // content collection, REQUIRED
  ev: 'click',  // sets the default event type to click
  animate: true, // slide animation?
  target: null  // target content to show on init
  trigger: 'a' // selector of each element within 'selector' that will trigger the event
});

NOTE: Options can be set using the a data attribute as well. For example if using an anchor, the href attribute is used to set the target, to add the collection you could do this:


  <a href="#foo" data-role="switcher" data-collection="switcher-content">Foo</a>
